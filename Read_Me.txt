Instructions to run this project :

1. Pre-Requisities:

Below are the libraries that needs to be installed in the project environment but not in the base environment of the acandonda.

Opencv
Tensorflow
Dlib
Keras
Numpy
Pygame

Supporting files for this project: 

haarcascade files are in xml docs folder, Trained model file are save in h5_file.

Drowsiness detection.py is the main execution file of this project.

CNN model training.ipynb is the model training file.

Output image is the image taken by our model when feeling drowsy.
